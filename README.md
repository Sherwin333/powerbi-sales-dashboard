# 📊 Sales Performance Dashboard

🚀 Interactive Power BI dashboard + Excel VBA automation to analyze and streamline retail sales reporting.

## 🧰 Tools & Technologies

* Power BI
* Microsoft Excel
* VBA (Visual Basic for Applications)

---

## 📌 Project Overview

This project presents a **4-page interactive Power BI dashboard** built to analyze retail sales performance across multiple dimensions including time, region, and product categories.

Additionally, an **Excel VBA automation tool** was developed to streamline monthly reporting by converting raw data into a structured report with a single click.

---

## 🚀 Key Features

### 📊 Power BI Dashboard

* KPI cards for **Total Sales, Units Sold, and Target Performance**
* Monthly **sales trend analysis**
* **Regional and category-wise breakdown**
* Interactive **slicers** for filtering data dynamically
* Conditional formatting to highlight performance insights
* Clean, professional layout inspired by real-world dashboards

---

### ⚙️ Excel VBA Automation

* Automated transformation of raw data into a formatted report
* One-click macro execution
* Reduced manual effort and improved reporting efficiency
* Structured output for consistent monthly reporting
